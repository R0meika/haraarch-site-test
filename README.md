# HARA Site Prototype

Первая статическая версия сайта для архитектурной и дизайн-практики HARA.

## Как открыть локально

1. Перейдите в папку `d:\Codex\hara-site`
2. Запустите:

```powershell
python -m http.server 8000
```

3. Откройте в браузере `http://localhost:8000`

## Состав

- `index.html` — структура страницы
- `styles.css` — визуальный стиль и адаптивность
- `script.js` — анимация появления блоков и демо-логика формы

## Что дальше

- заменить текстовые демо-кейсы на реальные проекты
- вставить фотографии, планы, визуализации
- подключить домен и настоящую почту
- при желании перенести в `Next.js`, `Webflow` или `Craft CMS`
